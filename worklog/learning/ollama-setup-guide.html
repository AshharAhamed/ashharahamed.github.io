Introduction

Ollama lets you download and run open-source Large Language Models (LLMs) directly on your own computer - no cloud account, subscription, or internet dependency required once a model is downloaded. This guide walks through the full process end to end: checking which models your hardware can realistically handle, choosing a model that fits your use case, installing Ollama, and running your first local chat session. This is the same setup used to run the LLM (Qwen2.5:0.5B) behind the RAG API with FastAPI project.

Who this guide is for: Anyone who wants to experiment with LLMs privately and locally, including developers, students, researchers, and hobbyists with no prior experience running AI models on their own machine.

Step 1 - Check Compatible Models Using llmfit

Not every model runs well on every machine. Larger models need more RAM and, ideally, a dedicated GPU. Before downloading anything, use llmfit to analyze your CPU, RAM, and GPU, and get a recommendation of which models will run smoothly on your specific hardware.

Install llmfit

Windows (Scoop)

scoop install llmfit

macOS (Homebrew)

brew install llmfit

Linux

# Global install
curl -fsSL https://llmfit.axjns.dev/install.sh | sh

# Local install (no sudo)
curl -fsSL https://llmfit.axjns.dev/install.sh | sh -s -- --local
Note: Always review install scripts from third-party sources before piping them into a shell, especially on shared or production machines.

Official resource: https://www.llmfit.org/

Step 2 - Find an Ollama Model

Browse the official Ollama Model Library to pick a model suited to your use case (general chat, coding, reasoning, etc.). Popular model families include:

Model Family Best Known For
Qwen Strong general-purpose and multilingual performance
Gemma Lightweight, fast, Google-developed models
Llama Widely supported, general-purpose Meta models
DeepSeek Strong reasoning and step-by-step problem solving
Mistral Efficient, high-quality open-weight models
Phi Compact Microsoft models, good on limited hardware
CodeLlama Code generation and completion
Tip: Cross-check the llmfit recommendation from Step 1 against the model sizes listed in the library so you pick something your hardware can run comfortably.

Official Model Library: https://registry.ollama.com/library

Step 3 - Install Ollama

  • Visit https://ollama.com/download
  • Download the installer for your operating system (Windows, macOS, or Linux).
  • Run the installer and follow the on-screen instructions to complete setup.

Step 4 - Pull a Model

Open PowerShell (Windows) or your terminal (macOS/Linux), then pull a model:

ollama pull qwen2.5:0.5b

More examples:

ollama pull gemma3:4b
ollama pull llama3.2:3b
ollama pull deepseek-r1:8b

Step 5 - Run the Model

ollama run qwen2.5:0.5b

Or:

ollama run gemma3:4b
Once running, the model loads into an interactive chat prompt directly in your terminal. Type your message and press Enter to chat.

Useful Ollama Commands

Purpose Command
List installed models ollama list
Show Ollama version ollama --version
Show help ollama --help
Remove a model ollama rm qwen2.5:0.5b
Stop a running model ollama stop <model>
Show model details ollama show <model>

Troubleshooting

Issue Likely Cause / Fix
Model runs very slowly Model is too large for available RAM/GPU - re-check with llmfit and try a smaller variant (e.g. 0.5b or 3b).
ollama command not found Restart your terminal after installation, or confirm Ollama was added to your system PATH.
Model download fails or stalls Check your internet connection and available disk space, then re-run ollama pull <model>.
Out of memory errors Close other memory-heavy applications, or switch to a smaller quantized model.

Workflow Summary

  • Install llmfit
  • Analyze your hardware
  • Choose a compatible model
  • Install Ollama
  • Pull the model - ollama pull <model>
  • Run the model - ollama run <model>
  • Start chatting with your local AI
